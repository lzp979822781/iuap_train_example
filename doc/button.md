# button使用

## 1 示例说明
     
本文档说明的是带图标的使用说明，为项目级button使用封装，使用示例以新增按钮如下

	<Button
		// 其余属性自定义
		iconType = 'uf-plus'
	>
		{item.text}
	</Button>

## 2 参数说明

配置带图标的的按钮,只需要配置iconType属性,其余属性参考官网[example link](http://bee.tinper.org/bee-button#bee-button)
